Tässä osiossa tutustutaan erilaisiin tiedonsiirtoprotokolliin ja niiden käyttötarkoituksiin. Samoin tutustuaan moniin dataformaatteihin, joissa dataa lähetetään ja talletetaan levylle esim. Big Data-järjestelmissä.

Opit tuntemaan eri data formaatteja ja tiedonsiirtoprotokollia. Opit tietämään niiden käyttötarkoituksia ja tekemään valintoja eri vaihtoehtojen välillä. 

Opit käyttämään pyarrow-kirjastoa ja tallettamaan sillä listan dataobjekteja niin Parquet kuin Orc-muotoihin. Opit hakemaan dataa Parquet-tiedostosta DuckDB-pyuthonkirjastolla.
